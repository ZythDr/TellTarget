# TellTarget
A tiny World of Warcraft **Vanilla 1.12.1** addon that makes whispering your current target faster.

## Features
- `/tt` whispers your current target
- Typing `/tt ` instantly expands to `/w TargetName `
- Works for any player target (even enemy faction players, TurtleWoW-friendly)
- Compatible with chat addons like [WIM](https://github.com/Kirchlive/WIM)

## Usage
Target a player and type: `/tt` and hit space

https://github.com/user-attachments/assets/220ead04-22e6-4533-8f15-d9281872c910

## Requirements
- World of Warcraft **Vanilla 1.12.1** (not Classic)

## Installation
1. Download ZIP from the green Code button above
2. Extract and move TellTarget to `WoWDirectory/Interface/AddOns/`
3. Confirm that `.toc` file is located at `WoWDirectory/Interface/AddOns/TellTarget/TellTarget.toc`
3. Restart WoW
